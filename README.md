# Sistema de Perfil Universitario — Symfony

Proyecto académico desarrollado con Symfony como parte de la materia de Programación Avanzada.

## Tecnologías Utilizadas

* PHP 8.x
* Symfony Framework
* Twig Template Engine
* HTML5
* CSS3
* Git y GitHub

## Funcionalidades

* Página principal de presentación
* Información personal del estudiante
* Listado de materias académicas
* Formulario de contacto
* Navegación entre secciones

## Instalación

1. Clonar el repositorio
2. Instalar dependencias

```bash
composer install
```

3. Ejecutar servidor local

```bash
symfony server:start
```

4. Abrir en el navegador

```text
http://127.0.0.1:8000
```
